Projects

Android App

Java | JavaFX | Android Studio | MVC
Using Android Studio I created an app for a pizza store to manage orders. This app was ported from a JavaFX application. The functionality includes placing and removing orders, calculating order cost, and visualizing store orders using a RecyclerView. Design patterns used for this app were Model View Controller (MVC) and Abstract Factory.

Temtem Viewer

React | Node | Express | MongoDB | TailwindCSS | REST API
View all temtem in the game and guess the daily temtem. See the 3d renders, stats, evolutions, and shinies of each temtem. Post to the discussion board about the temtem of the day and like or dislike others comments. Daily temtem changes every day. Link to repo

Inventory Management System - In Progress

React | Node | Express | MongoDB | TailwindCSS | REST API
This is a full stack web app that lets users interact with Inventory. This app uses Node.js, Express, and MongoDB for the backend, obeys RESTful API standards, and React.js in the front end. With a clean UI managers and customer service representatives can clearly and concisely find information on products in the store, see items on the way through the orders screen, and admins can order product.
